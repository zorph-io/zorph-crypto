# Security Policy

## Supported Versions

| Version | Supported          |
| ------- | ------------------ |
| 0.1.x   | :white_check_mark: |

## Reporting a Vulnerability

If you discover a security vulnerability in zorph-crypto, please report it responsibly.

**Do NOT open a public GitHub issue.**

Instead, email: **security@zorph.dev**

We will acknowledge receipt within 48 hours and aim to provide a fix or mitigation within 7 days for critical issues.

## Scope

This policy covers the `zorph-crypto` crate, including:

- AES-256-GCM encryption/decryption
- Key derivation (Argon2id)
- Ed25519 signing/verification
- BLAKE3 hashing
- X25519 key exchange
- Shamir Secret Sharing
- Multisig verification
- SEV-SNP attestation verification
